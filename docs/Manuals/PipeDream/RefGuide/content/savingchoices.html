Saving Choices

You can create a Choices file to ensure that every time you start up PipeDream, the settings you require are already in place. The Choices file is used each time you start PipeDream and applies to every PipeDream document.

The following settings are saved in the Choices file:

  • Font information
  • Line separator (Save dialogue box)
  • Printer Config dialogue box information
  • Page Layout dialogue box information
  • Options dialogue box information.

These settings can also be saved in template files.

  • Short / long menus
  • Menu configuration
  • Colours
  • Automatic Check Status (for spelling)
  • Open user dictionaries
  • Size of paste list
  • Recalculation state: auto / manual

These settings are independent of individual documents and are all set from the Choices sub-menu.

Creating a Choices File

  1. Click the PipeDream 4 icon on the icon bar to open a new PipeDream document.
  2. Enter the settings you prefer in the appropriate menus and dialogue boxes.
  3. When all the settings are as you want them, give the Save Choices command:
Menu Command Tool Key Cmd-sequence
Icon Save choices FGI

From now on, the settings you have saved will be there whenever you start PipeDream.

When PipeDream starts, it looks for a Choices file and, if there is one, loads the settings.

Template Files

You can also create template files. Template files do not contain all the user interface settings of the Choices file. They contain the starting points for different types of document you may wish to create. If you load a template file to start a new document, the file-specific information overrides that in the Choices file.
