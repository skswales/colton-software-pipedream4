Icon Menu

If you move the pointer to the PipeDream 4 icon on the icon bar and click Menu, you will see PipeDream’s Icon menu.

Info tells you your PipeDream version number and, if you have upgraded from a paid-for copy of PipeDream 4, user name, organisation name and registration number.

Documents displays a list of windows currently open. Click Select on a document’s name to bring it to the front.

New document displays a list of template files which you can select to start a new document.

Choices displays the Choices sub-menu which contains all the options you can set to apply to PipeDream documents. These settings can be saved with the Save Choices command.

Save choices enables you to save the settings made on the Choices sub-menu in a Choices file so that those settings will be there automatically for each new document.

Quit leaves PipeDream, removing the PipeDream 4 icon from the icon bar. All documents in memory and all open dictionaries will be closed. You will be asked whether you want to save any modified documents. If so, click the Save button. If not, click the Discard button. If you change your mind about quitting, click the Cancel button (or press Escape). You will also be given these options when the RISC OS desktop is closed down when there are modified documents in memory.
