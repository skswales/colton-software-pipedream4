ViewSheet File Import

To transfer a ViewSheet file to PipeDream, drag its file icon from a directory display to the PipeDream 4 icon on the icon bar. It is recognised and loaded automatically. Do not double-click the ViewSheet file icon.

The conversion from ViewSheet to PipeDream includes all ranges and functions.

Ranges on their own are converted to SUM() functions, for example, A1A10 would become SUM(A1A10).

Text cells retain the justification they had in ViewSheet.

For number cells, the decimal places, minus / brackets and justification settings are retained at cell level but not at window level.

The following ViewSheet items are not converted:

  • windows;
  • printer windows;
  • row and column headings;
  • protection;
  • column widths.
