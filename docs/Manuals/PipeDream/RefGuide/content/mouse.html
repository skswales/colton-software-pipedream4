Mouse Techniques

The following is a summary of mouse operations which apply to a PipeDream window.

Positioning the Caret

Moving the pointer onto a row and clicking Select usually positions the caret in the cell containing the pointer. However, if that cell is blank, and there is a text cell to the left which can overlap the blank cell, PipeDream puts the caret in the text cell. This is to prevent you from accidentally typing text into two different columns on the same row. If you really want the caret to be in the blank cell into which you moved the pointer, hold down Ctrl and then click Select.

Alternatively, click Select on the heading of the column and PipeDream will move the caret along the row into that column.

Loading Documents

To load a PipeDream document, either double-click its file icon in a directory display, or drag the file’s icon from a directory display (or other running application) onto the PipeDream 4 icon on the icon bar.

Inserting Documents

Drag a file’s icon from a directory display (or other running application) into a window to insert the file at the caret position.

Creating a PipeDream Document

To create a new PipeDream document:

  1. Click the PipeDream 4 icon on the icon bar to bring up the New document dialogue box;
  2. Click the up / down arrows to find the template file you want;
  3. Click the Create button (or press Return↵).

Bringing up Menus and Choosing Commands

With the pointer in a PipeDream document window, click Menu to bring up the top level menu.

To bring up a sub-menu, slide the pointer over the arrow to the right of the menu name.

To choose a command, click Select on the command name. This carries out the command and removes the menu from the screen. If you click Adjust, the menu remains on the screen.

To repeat the last command, use the Repeat Command command:

Menu Command Tool Key Cmd-sequence
Files Repeat command F5 J

This enables you to repeat commands easily without having the menu in the way or having to type long Ctrl-key sequences.

Putting Cell References Into the Formula Line

To put a cell reference into the formula line or formula window whilst you are editing a formula, click Select in the cell to which you want to refer. The cell can be in the current window or in another window.

By default, relative cell references will be inserted. To fix the row part of the reference, hold down the Shift key when you click in the cell. To fix the column part of the reference, hold down the Ctrl key when you click in the cell. PipeDream will add $ signs to fix the reference.

To insert a range in the formula line or formula window you can either click Select at the start and the end of the range, or you can drag from the start to the end of the range.

Using Scroll Bars

To scroll one line in the direction of the arrow, position the pointer on the scroll arrow and click Select. Hold down Select to continue scrolling. Click Adjust to scroll back the other way. The caret always remains within the PipeDream document window, staying as near as possible to its original position.

To scroll windowful by windowful, position the pointer on the darker part of the scroll bar (above the slider to scroll up, below the slider to scroll down) and click Select. The next windowful of the document will appear. Click Adjust to scroll back the other way. The caret always remains within the PipeDream window, moving, if necessary to the point nearest to its position before scrolling.

To move down through the document, position the pointer on the slider, press Select and hold it down. Drag the slider to display the part of the document you want to see.

To move from one side of a document to the other, use the horizontal scroll bars. The caret always remains within the PipeDream document window, staying as near as possible to its original position.
