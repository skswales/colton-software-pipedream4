Functions

PipeDream provides many standard functions to help you to perform calculations within number cells. Most functions take parameters, i.e. they perform calculations on numbers, dates, strings, etc., which you specify. The following is a brief list of all PipeDream’s standard functions. For a detailed description of any individual function, refer to the relevant section.

PipeDream also enables you to create your own functions. See Custom Functions.
