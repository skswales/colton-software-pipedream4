Printing

To Print a Document Using a RISC OS Printer Driver

First ensure that a RISC OS printer driver is loaded. Your RISC OS printer driver icon must be on the icon bar.

  1. Give the Printer Config command:
Menu Command Cmd-sequence Key
Print Printer config PD
  • 2. Set the Printer type option to RISC OS. Ignore the other options.
  • 3. Click the OK button or press Return↵.
  • 4. Give the Print command.
Menu Command Tool Key Cmd-sequence
Print Print Print PO

You will see the Print dialogue box. At the top of the dialogue box, you will see the type of RISC OS printer driver currently available.

  • 5. If you want to print more than one copy of the document, specify the number of copies.
  • 6. Select the Landscape to print the document horizontally or Portrait to print it vertically. If you decide to print your document in landscape format, remember that the page length will change as a result. If you do not have the Auto line height option switched on, you will need to tell PipeDream the new page length in the Page Layout dialogue box. Try about three quarters of the portrait page length as your new page length. For example, if the portrait page length is 66, try 48. You may also need to tell PipeDream the new page width. Try the portrait page width plus a third. For example, if the portrait page width is 80, try 106. If the document you are printing is to be bound as a book, select the Two sided, margin option to print out your document with the headers and footers reversed on left-hand pages and with extra characters added to the right-hand pages’ left margins to leave space for the binding. Specify the number of characters you want to add to the margins on the right hand pages.
  • 7. Select the Wait between pages option if you want to wait after each page to insert single sheets of paper. For each page, you will be given the following options:
    • miss out the page;
    • print to the end of the document;
    • print the current page;
    • stop printing.

To Print a Document Using a PipeDream Printer Driver

  1. Give the Printer Config command:
Menu Command Cmd-sequence Key
Print Printer config PD
  • 2. Select the PipeDream printer driver you are going to use by clicking on the up / down arrow buttons. If you have loaded a PipeDream printer driver already, its name will appear here. Printer drivers are provided for dot matrix printers, daisy wheel printers and HP PCL printers. These are called DotMatrix, Daisywheel and HPJet and are found in the !PipeDream.Resources directory.
  • 3. On the Printer type line, specify whether you have a serial or parallel printer connected by using the up / down arrow buttons to set this option to Serial or Parallel. Most printers require the Parallel setting. Set the option to Network if you want to use an Econet printer server. If you do, type in the number or name of the printer server in the Printer server input field. The remaining options apply only to serial ports. The default settings will probably work; if not, refer to your printer’s user guide for the settings you should use.
  • 4. When you have set the configuration, click the OK button or press Return↵.
  • 5. Give the Print command:
Menu Command Tool Key Cmd-sequence
Print Print Print PO

You will see the Print dialogue box.

  • 6. If you want more than one copy of the document, type in the number of copies you require.
  • 7. Select the Printer option to ensure that the file is sent to your printer. This is the default option.
  • 8. Select the Wait between pages option if you want to wait after each page to insert single sheets of paper. For each page, you will be given the following options:
    • miss out the page;
    • print to the end of the document;
    • print the current page;
    • stop printing.

To Print Part of a Document

To print a marked block:

  1. Mark the block you want to print.
  2. Give the Print command.
  3. When the dialogue box appears, select the Print marked block option.

To See the Results of Printing on the Screen

This option is only useful for seeing where headers and footers appear.

  1. Give the Print command.
  2. When the dialogue box appears, select the Screen option.

You will see your document on the screen including headers and footers. Fonts are not displayed.

To Print to a File

If you are using a RISC OS printer driver, click Menu on the printer icon on the icon bar and type in the name of the file to which you want to print.

If you are using a PipeDream printer driver, select the File option in the Print dialogue box and type in the name of the file to which you want to print. Printer codes from the printer driver are included in the file.

To Cancel Printing

You can cancel printing by pressing Escape. Text already in the print buffer may be printed.
