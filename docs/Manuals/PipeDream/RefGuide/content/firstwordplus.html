First Word Plus File Transfer

PipeDream can load and save files in First Word Plus format. As with most file conversion between different applications, it is not possible to perform a perfect conversion because the two programs have different philosophies and structures. However, the converter does ease the task of passing text between the two applications considerably.

Loading a First Word Plus File

To load a First Word Plus file, drag the file’s icon from its directory to the PipeDream 4 icon on the icon bar. All First Word Plus files have a recognisable header at the beginning, so the PipeDream auto-loader should have no trouble determining that the file is a First Word Plus file.

PipeDream extracts all the text and most of the highlighting information from the First Word Plus file. It should also recover some of the page layout information such as the page length, margins, header and footer.

After loading the file, you will probably want to save it in PipeDream format. It is best to save the PipeDream document immediately. Give the Save as command, change the filename, select the PipeDream format option and then click the Save button (or press Return↵).

Saving a First Word Plus File

To save a file in First Word Plus format, give the Save as command. Check that the filename is what you need. You will probably not want to save your First Word Plus file on top of the PipeDream file that you loaded. Select the FWP format option. Click the Save button (or press Return↵). The file will be saved in First Word Plus format.

PipeDream saves out all the text and all your highlights. It also tries to send out page layout information such as page length, margins, header and footer. PipeDream sends column information out separated by Tabs. Note that, because First Word Plus cannot store and save Tabs (it converts them to spaces), you will not be able to edit the file in First Word Plus, reload it into PipeDream and maintain all the column information.
