Protection

There may be some cells in your PipeDream document which you do not want to be altered either deliberately or accidentally. For example, you may have set up a table for somebody else to update. You want to be sure that the permanent information is not altered. You can protect either individual cells or blocks of cells using the Set Protection command:

Menu Command Tool Key Cmd-sequence
Blocks Set protection BPS

If you have marked a block of cells, all cells in the block except blank cells will be protected. Otherwise the current cell will be protected.

The protected cell or cells can be displayed with a different background colour, so that you can easily see which cells are protected. You can specify the colour in the Colours dialogue box.

To remove protection, give the Clear Protection command:

Menu Command Tool Key Cmd-sequence
Blocks Clear protection BPC

Protection will be removed from the marked block of cells, if there is one, or from the cell containing the caret.

To remove all protection from a document, mark the whole document by clicking the mark block box before giving the Clear Protection command.
