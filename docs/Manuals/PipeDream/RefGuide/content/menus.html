Menus

PipeDream provides the following menus accessed by clicking Menu in a document window:

  • Files
  • Edit
  • Layout
  • Print
  • Blocks
  • Caret
  • Spell

The following menus may be accessed by clicking on toolbar buttons in a document window:

  • Charts
  • Functions

The following menus may be accessed by clicking Menu on the PipeDream icon:

  • Icon
  • Choices sub-menu

Each consists of a list of commands, some of which have dialogue boxes. When you start PipeDream, the menus you see are short menus, which display those commands which are generally most frequently used. If you switch to long menus, you can see all the commands for each menu.

Customising Menus

If the commands on any short menu are not those which you personally use most frequently, you can customise the menus to suit your own needs. Before customising the menus, decide which commands you want to move and make a note of their Ctrl-key sequences. You will need to switch on long menus to make a note of the Ctrl-key sequences of commands on long menus or refer to the menus shown above and compare them to your existing short menus.

Removing a Command from a Short Menu

Example:

To move the Load command from the short Files menu to the long menu, proceed as follows:

  1. Give the Change Menus command:
Menu Command Tool Key Cmd-sequence
Choices Change menus FGM
  • 2. Give the Load command by pressing Cmd-FL or by choosing the Load option from the Files menu.

The Load command will not be carried out but will be moved from the short menu to the long menu. You will see that the Load command is no longer on the short Files menu.

To Move a Command to a Short Menu

Example:

To move the Load command back from the long Files menu to the short menu, proceed as follows:

  1. Give the Change Menus command.
  2. Give the Load command by pressing Cmd-FL.

The Load command will not be carried out but will be moved from the long menu to the short menu. If you switch off long menus, you will see that the Load command is now back on the short Files menu.

When you have customised the menus to suit your needs, use the Save Choices command to save your chosen menu configuration.

In this way, you can ensure that you have all the commands you use most often readily available on short menus and those commands you rarely use on long menus.
