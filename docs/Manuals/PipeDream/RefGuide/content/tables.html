Creating Independent Columns of Text

By default, the Insert on wrap option in the Options dialogue box is set to Rows, to ensure that information in adjacent columns remains correctly aligned when text is inserted within any column. You can also create independent columns of text in one PipeDream document, like a newspaper. Setting the Insert on wrap option to Columns makes all columns independent, so that inserting text within one column will not affect columns to either side of it. The Insert on wrap setting is preserved when you save a document.

To create a multi-column layout, first design the document. For example, to create a layout with three equal columns, set the width to 24 and the right margin to zero for all three columns. To do this, use the Set Column Width and Set Right Margin commands. The Set Column Width command enables you to set the width for all the columns. The Set Right Margin command enables you to specify the position of the right margins for the columns. This determines the point at which the column will wrap. You do not need to carry out these commands three times. Instead, you can specify a column range A C, which will cause the command to be applied to columns A, B and C.
