Commands

This alphabetical list summarises all the PipeDream commands, with the following information:

Command

This gives the command name as shown on the menu.

Menu

This tells you in which menu or sub-menu the command is found. Some PipeDream commands do not appear on menus because they are always given by clicking the mouse or pressing keys or because they are only used in command files.

The Choices sub-menu is available from the Icon menu.

The Function menu is displayed by clicking the Function toolbar button.

L/S

This tells you whether the command is found on long or short menus and applies to the menus reached via the PipeDream main menu. Other menus always show all their menu options.

Key

This gives the function key or other keyboard shortcut to which the command is assigned.

Cmd-sequence

This tells you the classic key sequence combination for the command.
If you are not using Classic menus, press Ctrl-\ (Ctrl-Backslash) followed by cmd. Note that Ctrl need only be held down for the initial backslash.
If you are using Classic menus, press Ctrl-cmd. Note that Ctrl need only be held down for the initial letter of the command.

Section

This gives the name of the section in the Reference Guide in which the command is explained in detail.

As you can see, PipeDream provides several methods of giving commands. It is entirely up to you which method you use. Some people use one method all the time; others use a combination of all of them.

Throughout this Reference Guide and the Getting Started Guide, the various methods will generally be shown as follows and it is up to you which method you use:

  2. Move the pointer onto the File option and then slide it over the arrow to display the File menu.
  3. Click the Save command.
