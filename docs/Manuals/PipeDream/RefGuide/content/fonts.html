Fonts

Many printers are able to print your text in a range of different fonts. You can also see fonts displayed on your screen. A font is the name given to a design of character (typeface) presented at a given weight, style and size.

You can use fonts to make your documents appear much more stylish and professional. You might want to have your headings printed in one font and your main text in another or you might want to pick out certain words or phrases by printing them in a different font to that used for the rest of the text. In this Reference Guide, for example, the main text, such as this paragraph, is in a sans-serif typeface and text which appears on the screen is shown in a monospace typeface.

With PipeDream, you can use any outline font which is compatible with RISC OS. Fonts are displayed on the screen and can be printed out on a suitable printer using a RISC OS printer driver.

Setting a Global Font

The Font command sets a global font for the document. Note that you will only be able to print the font if you use a RISC OS printer driver.

  1. Move the pointer over the arrow to the right of Font in the Print menu to see the Font Selector dialogue box.
  2. Click the name of the Font you require. You may have to use the up / down arrow buttons to scroll through the list of fonts to find the one you want. The weights and styles available for the font you have chosen will be displayed.
  3. Select the Weight you require by clicking on its name.
  4. Select the Style you require by clicking on its name.
  5. Select the Point size you require by clicking it. If you want to, you can change either the height or width by setting it to what you require with the up / down arrow buttons.
  6. If you want to see what the font will look like, click the Try button. The alphabet will be displayed in the font you have chosen. You can type in other characters to see what they look like in your chosen font.
  7. Set the Line height you require. Font sizes and line height settings are rather subjective but having a Line height about 120% of the font height usually gives a pleasant effect. By default, the line height is set to Auto which means that PipeDream will give you 120% of the current height setting. You can change this if you want to do so.
  8. To select this font, click the Set button (or press Return↵)

Note that if you select a very small font, PipeDream will display it on the screen within PipeDream’s minimum screen line height. It will be printed with the correct line height for the font you have selected.

Specifying a Change of Font

The Local Font command enables you to specify a local change of font for a cell. At the end of the cell, PipeDream switches back to the font you have set for the whole document with the Font command. Note that you can only have a local change of font if you have already selected a global font using the Font command.

Example:

@F:Corpus.Medium,24,24@Chapter Ten

would print your chapter title in 24 point Corpus.Medium.

To specify a change of font, proceed as follows:

  1. Position the caret at the beginning of the cell you want to change.
  2. Bring up the Print menu.
  3. Move the pointer over the arrow to the right of the Local Font option in the Print menu to see the Local Font dialogue box.
  4. Click the name of the Font you require.
  5. Select the Weight you require by clicking on its name.
  6. Select the Style you require by clicking on its name.
  7. Select the Point size you require by clicking on it. If you want to, you can change either the height or width by setting it to what you require with the up / down arrow buttons.
  8. If you want to see what the font will look like, click the Try button. The alphabet will be displayed in the font you have chosen. You can type in other characters to see what they look like in your chosen font.
  9. To insert the local font change, click the Insert button (or press Return↵)
