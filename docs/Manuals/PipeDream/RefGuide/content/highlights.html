Highlights

Highlights enable you to exploit the special facilities of your printer to print text in different styles.

The following highlight options are available from the Print menu. You can either select them from the menu or type in the key combinations.

Highlight Effect Tool Key Cmd
seq
H1  Underline PU
H2 Bold Ctrl-B PB
H3 Extended sequence  PX
H4 Italic Ctrl-I PI
H5 Subscript PL
H6 Superscript PR
H7 Alternate font PA
H8 User-defined PE

The effect of the bold, underline, italics, superscript and subscript codes are displayed on the screen. The other codes are displayed as highlighted numbers. The effect of these codes when printing the document will depend on the facilities of the printer driver being used.

To highlight a small section of text

Insert the appropriate highlight codes each side of the text to be highlighted.

For example, to underline a word:

  1. Position the caret at the start of the word and give the Underline command:
Menu Command Tool Key Cmd-sequence
Print Underline PU
  • 2. Position the caret at the end of the word and give the Underline command again.

To put a word in bold:

Give the Bold command at the start and end of the word:

Menu Command Tool Key Cmd-sequence
Print Bold Ctrl-B PB

In the case of bold and underlined text, the effect will be displayed on the screen when you move the caret off the cell. Note that highlights are automatically cancelled at the end of a cell.

To highlight a block of text

  1. Mark the block.
  2. Give the Highlight Block command:
Menu Command Tool Key Cmd-sequence
Print Highlight block PHB
  • 3. When the dialogue box appears, type in the required highlight number.

Typing in 1, for example, would cause every word in the marked block to be underlined.

When a block is marked, clicking one of the highlight buttons on the toolbar will execute the Highlight Block command.

To remove highlights from a block of text

  1. Mark the block.
  2. Give the Remove Highlights command:
Menu Command Tool Key Cmd-sequence
Print Remove highlights PHR
  • 3. When the dialogue box appears, type in the number of the highlight to be removed from the block.
