Alignment

Default Alignment

Text typed into text cells is normally aligned with the left-hand margin. The values of number cells are normally displayed right-aligned within the column width. For tables of figures displayed with the same number of decimal places, this ensures that the decimal points line up.

Altering Alignment

The default alignment of both text cells and number cells can be changed. You can choose to have the values of number cells left-aligned or centred over the column. You can choose to have text centred, right-aligned or sections of the line aligned in all three ways.

With text cells, the effect can only be seen when the caret is not in the cell. When the caret is in the cell, the text is displayed left aligned. This makes the editing of text cells easier.

  1. Mark the cell or block of cells whose alignment you want to change.
  2. Give one of the following commands:
  • Left Align to align the text or value with the left-hand margin.
  • Centre Align to centre the text or value.
  • Right Align to align the text or value with the right-hand margin.
  • LCR Align to align text left, centred and right on the same line (text cells only). The first character on the line is taken as a delimiter to separate the line into three sections. The first section is left aligned, the second is centred and the third is right aligned.
  • Free Align to remove any previous alignment. This resets text cells back to left alignment and number cells back to right alignment.

The commands are given as follows:

Menu Command Tool Key Cmd-sequence
Layout Left align LAL
Centre align LAC
Right align LAR
LCR align LLCR
Free align LAF
