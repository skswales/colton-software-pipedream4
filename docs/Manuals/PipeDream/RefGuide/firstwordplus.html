First Word Plus Format

PipeDream can load and save files in First Word Plus format. As with most file conversion between different applications, it is not possible to perform a perfect conversion because the two programs have different philosophies and structures. However, the converter does ease the task of passing text between the two applications considerably.

Loading a First Word Plus File

To load a First Word Plus file, drag the file's icon from its directory to the PipeDream icon on the icon bar. All First Word Plus files have a recognisable header at the beginning, so the PipeDream auto-loader should have no trouble determining that the file is a First Word Plus file.

After loading the file, you will probably want to save it in PipeDream format. It is best to save the file immediately. Bring up the Save dialogue box, change the filename, click Select on the PipeDream format option and then click the OK button or press Enter.

PipeDream extracts all the text and most of the highlighting information from the file. It should also recover some of the page layout information such as the page length, margins, header and footer.

Saving a First Word Plus File

To save a file in First Word Plus format, first bring up the Save dialogue box. Check that the filename is what you need. You will probably not want to save your First Word Plus file on top of the PipeDream file that you loaded. Click Select on the 1WP format option. Click the OK button or press Enter. The file will be saved in First Word Plus format.

PipeDream saves out all the text and all your highlights. It also tries to send out page layout information such as page length, margins, header and footer. PipeDream sends column information out separated by Tabs. Note that, because First Word Plus cannot store and save Tabs (it converts them to spaces), you will not be able to edit the file in First Word Plus, reload it into PipeDream and maintain all the column information.
