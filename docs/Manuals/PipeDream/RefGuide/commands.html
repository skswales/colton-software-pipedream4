Commands

This alphabetical list summarises all the PipeDream commands, with the following information:

Command

This gives the command name as shown on the menu.

L/S

This tells you whether the command is found on long or short menus and applies to the menus reached via the PipeDream main menu. Other menus always show all their menu options.

Menu

This tells you in which menu or sub-menu the command is found. Some PipeDream commands do not appear on menus because they are always given by clicking the mouse or pressing keys or because they are only used in command files.

The Choices sub-menu is available from the Icon menu.

The Function menu is displayed by clicking Select on the Function button.

Ctrl-sequence

This tells you the Ctrl-key sequence combination for the command.

Key

This gives the function key or other keyboard shortcut to which the command is assigned.

Section

This gives the name of the section in the Reference Guide in which the command is explained in detail.
