Page Layout

The Page Layout dialogue box enables you to change the page length, margins and line spacing of your document. You can also use it to set up headers and footers, to specify at which number page numbering should start and to set the page width to which PipeDream should work.

The default settings are as follows:

Page length 66
Top margin 0
Header margin   2
Footer margin 2
Bottom margin 8
Left margin 0
Page width 80

Headers, Footers and Page Numbering

You can define a header and a footer for each document. These will be printed out at the top and bottom of each printed page. Headers and footers can include text @-fields. Thus the page number (@P@), date (@D@) or document title (@T@) can be included in the header or footer text. Headers and footers can also include bold or underlined text, or any of the other highlight codes. Headers and footers can be up to three lines long.

Remember that having headers and footers reduces the lines available for your text.

To define a header or footer

In the Page Layout dialogue box, you will see that there are three lines for you to type your header and footer. Headers and footers can be one, two or three lines long. Specify each line of the header and footer as three fields, separated by a character such as /, on the Header or Footer line of the dialogue box.

For example:

/left/centre/right/

Note that you can use another character instead of / if you prefer but it must not occur elsewhere in the header or footer and must appear as the first character.

The fields are printed respectively left-justified, centred and right-justified on the page.

Note that PipeDream centres and right justifies with respect to the right-most right margin point of all the columns, so, if your header or footer appears in the wrong place, you need to adjust the right margin for the offending column.

Examples:

If you type your header text as follows:

/Colton Software///

/Tutorial/Chapter 1/Word Processing

///Draft only

Your header will appear as follows:

Colton Software
Tutorial Chapter 1 Word Processing
Draft only

If you want to include a blank line in your header or footer, you need to type delimiters with nothing between them (///) onto that line in the dialogue box. Completely empty lines are ignored.

To include page numbering

The header and footer give you a great deal of flexibility in the way you choose to number pages.

For example, to number pages in the form

-23-

centred at the bottom of each page, define the footer as:

//-@P@-//

To include highlights, type ^n where n is the highlight number. ^2, for example, is bold.

To control page numbering

You can choose the number for page numbering to start on any file with the Page Layout dialogue box option Start page. This can be set to any number. If you leave it blank, it will automatically start at page 1. If you set it to 0, the first page will be printed with no header or footer. This can be useful for title pages when you do not want a page number on the first page of the document. The next page will be page 1.

Page Width

You can set a page width to which PipeDream should work which will be determined by your printer and the paper size you are using. This option is useful when you are printing very wide documents such as large spreadsheets or databases.
