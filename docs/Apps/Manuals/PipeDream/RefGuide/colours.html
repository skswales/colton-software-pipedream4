Colours

You can select the colours you want to appear in your PipeDream windows.

1. Give the Colours command:

Menu Command Ctrl-sequence Key
Choices Colours

2. When the dialogue box appears, set the colours you prefer for the various parts of the PipeDream display - note that you may need to scroll the window to see all the options.

The numbers 0 to 15 refer to the colour settings on the palette. If you wish, you can change the settings on the palette. See your RISC OS User Guide.
