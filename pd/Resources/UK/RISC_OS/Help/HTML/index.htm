Support

As PipeDream 4 is now completely free to use, support is a self-help affair. There is a Google Group colton-software-pipedream where you may seek help from other users:
web site: https://groups.google.com/group/colton-software-pipedream
e-mail: colton-software-pipedream@googlegroups.com
Please include files which demonstrate any problems (using a Spark or ZIP archive).
