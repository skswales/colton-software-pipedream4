PipeDream - Spreadsheet Functions

PipeDream provides many standard speadsheet functions to help you to perform calculations within number cells. Most functions take parameters, i.e. they perform calculations on numbers, dates, strings, etc., which you specify.

The following is a brief list of all PipeDream’s standard speadsheet functions. For a detailed description of any individual function, refer to the relevant section below. Fuller details may be available for certain complicated functions in the Reference Guide.
