<h2>Items of historical interest</h2>

1992-05-22 <a href="/coltsoft-downloads/pipedream/Historic/PD_Impres_19920522_102.zip">PD_Impres_19920522_102</a>

This ZIP archive contains the PipeDream to Impression DDF file converter.

It is a 26-bit application and has been observed to work on 32-bit RISC OS under Aemulor Pro. YMMV.

1989-07-21 <a href="/coltsoft-downloads/pipedream/Historic/Z88FS_19890721_100.zip">Z88FS_19890721_100</a>

This ZIP archive contains the Z88: filing system and desktop filer module to allow files to be transferred to RISC OS from Z88 PC-LINK and vice versa.

It is a 26-bit module and is likely only to work on older 26-bit RISC OS versions. It hasn't been tested since the mid-1990s!

The Z88 lives on at <a href="http://www.rakewell.com">rakewell.com</a> - please do visit their site!
